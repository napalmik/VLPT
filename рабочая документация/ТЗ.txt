## 1. Общее описание

Локальное веб-приложение для учета учебной нагрузки преподавателей, управления расписанием и генерации отчетов (PDF/Word).  
Развертывание: локальный компьютер, доступ через браузер в локальной сети.

Ключевые функции:
- Интерактивный календарь с расписанием занятий.
- Личный кабинет преподавателя с авторизацией.
- Автоматический учет учебной нагрузки.
- Генерация отчетов (PDF, Word).

## 2. Текущие наработки (черновик)

- **Стек**: React + TS + Tailwind (frontend), Node.js + Express (backend), MySQL (Docker), Docker Compose.
- **БД (черновой вариант)**:
  - `teachers(id, email, password_hash, name, subject_id, workload_hours, access_level)`
  - `subjects(id, name)`
  - `groups(id, name)`
  - `schedule(id, teacher_id, group_id, subject_id, date, time, duration_hours, room)`
  - `reports(id, type, data, generated_at)`
- **Тестовые данные**:
  - Группа: `OSP(9)`
  - Преподаватель: `teacher@example.com / password123`
- **Идеи интерфейса**:
  - Календарь в стиле расписания: дни недели по горизонтали, группы по вертикали (на тестах — `OSP(9)`).
  - В кабинете преподавателя в правом верхнем углу — текущая нагрузка с подсветкой превышения.
